# KeenKeeper

## Project Description
KeenKeeper is a personal relationship management application designed to help users stay connected with the people who matter most. It allows you to track interactions, monitor friendship health, and ensure that no important relationship is neglected over time.

---

## Technologies Used
- Next.js
- React
- Recharts
- Tailwind CSS
- Vercel

---

## Key Features

### Friendship Health Tracking
Each connection is assigned a status such as On Track, Almost Due, or Overdue, allowing users to quickly identify which relationships need attention.

### Interaction Statistics
Provides visual insights through charts, including a pie chart representation of communication patterns such as calls, texts, and video interactions.

### Timeline View
Displays a chronological record of all interactions, helping users review their communication history and maintain meaningful relationships more effectively.
